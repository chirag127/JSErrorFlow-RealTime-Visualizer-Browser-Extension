About

JavaScript Error Visualizer is a browser extension that significantly improves the JavaScript debugging experience for web developers by providing immediate visual feedback directly on the webpage, pinpointing the DOM elements associated with runtime JavaScript errors.

No more context-switching between the webpage and the browser's developer console. See exactly where errors are occurring and get detailed information with just a glance.

Installation

1

Clone the Repository

git clone https://github.com/chirag127/JavaScript-Error-Visualizer-browser-extension.git
2

Install Dependencies

cd JavaScript-Error-Visualizer-browser-extension
npm install
3

Build the Extension

npm run build
4

Load the Extension in Your Browser

Chrome/Edge: Go to chrome://extensions/, enable Developer mode, click "Load unpacked", and select the dist folder.

Firefox: Go to about:debugging#/runtime/this-firefox, click "Load Temporary Add-on", and select any file in the dist folder.

How to Use

Enable/Disable

Click the extension icon and use the toggle switch to enable or disable the extension.

View Errors

When an error occurs, the associated DOM element will be highlighted on the page. Click the extension icon to see a list of all errors.

Error Details

Click on an error in the list to see detailed information, including the full stack trace.

Navigate to Elements

Click on the element links in the error details to scroll to and flash the highlighted element on the page.

Clear Errors

Use the "Clear All" button to remove all error highlights and reset the error list.

Settings

Click the "Settings" button to access the extension settings, where you can customize the highlight style, domain settings, and more.
